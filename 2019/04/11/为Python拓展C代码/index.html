为每一个模块函数添加形如 PyObject\ Module_func()*的封装函数

对于每个需要在Python环境中访问函数，需要创建一个以static PyObject*标识，以模块名开头，紧接着是下划线和函数名本身的函数。

例如，若要让fac()函数可以在Python中导入，并将Extest作为最终的模块名称，需要创建一个名为Extest_fac()的封装函数。

封装函数的任务是将Python中的值转为C形式，接着调用相应的函数。当C函数执行完毕时，需要返回Python环境中。封装函数需要将函数返回值转为Python形式，并进行真正的返回。

那么如何进行这种转换呢？

  • 从Python到C时，调用一系列PyArg_Parse*()函数
  • 从C到Python时，调用Py_BuildValue()函数
